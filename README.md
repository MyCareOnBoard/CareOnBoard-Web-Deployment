# Care-On-Board

Powering the delivery of Quality Care.

## Getting Started

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:3000`.

## Tech Stack

- Next.js 15
- React 19
- TypeScript 5
- Tailwind CSS 4 + `tw-animate`
- shadcn/ui components
- Lucide React icon set
